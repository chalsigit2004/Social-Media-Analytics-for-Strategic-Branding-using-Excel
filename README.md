# Social Media Analytics for Strategic Branding — Excel

## 📋 Business Question
Brands need to know which social content is actually driving engagement, not just guess. This project builds an Excel-based intelligence solution to clean, transform, and analyze social media metrics into a report a marketing team can act on.

## 🛠️ Tools Used
- **MS Excel** — pivot tables, formulas, data transformation
- **Reporting** — engagement and brand-performance summaries

## 🔍 Approach
1. Cleaned raw social media metrics (imports, formatting inconsistencies, duplicate entries)
2. Transformed the data into a structured format for analysis
3. Built pivot tables and summary views to track engagement trends over time
4. Produced a report highlighting brand performance patterns for marketing decision-making
